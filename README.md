## Description

RAS MarkfeederSystem is a modern web-based School Marks Management System designed for CBSE and State Board schools.

### Features

- Admin Login
- Teacher Login
- Student Management
- Teacher Management
- Marks Entry
- Grade Entry
- Report Card Generation
- Excel Export
- PDF Printing
- Analytics Dashboard

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
- PostgreSQL
- SheetJS
- Chart.js
